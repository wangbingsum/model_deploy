## 项目管理

    - pip安装国内源

        pip install 包名 -i https://pypi.tuna.tsinghua.edu.cn/simple

    - trt模型导出

        trtexec --onnx=./model/resnet50.onnx --saveEngine=./model/resnet50_fp16.engine --fp16

    - trt模型运行(加载测试)

        trtexec --loadEngine=.\model\resnet50_fp16.engine
